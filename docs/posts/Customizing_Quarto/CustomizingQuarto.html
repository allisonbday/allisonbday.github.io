Introduction

Your blog should show your personality. When I started to take blogging seriously, I wanted to customize my blog to make it more personal. I wanted to change the theme, colors, and fonts.

Pages

When you create a base Quarto blog, you get a couple of pages: an About page and an index page. I wanted to add a series page and a landing page too.

About Page

After filling in the about.md file by adding a bit about me and my education and experience, I wanted to do more customization.

Apps

Since Quarto is built on top of R Markdown, anything that can be added to markdown is available in Quarto! This includes some fun Github readme-stats apps.

First, I added a list of icons for my most used skills using the skill-icon app. I also wanted them centered, so I added a bit of html to the code.

<p align="center">
  <a href="https://skillicons.dev">
    <img src="https://skillicons.dev/icons?i=py,r,md,github,vscode" />
  </a>
</p>

I also wanted some of my Github Stats, so I added the github-readme-stats. You can replace <username> with your Github username. These come with lots of customization, but I like how their basic layout looks. (To get the apps side by side, you put them on the same line)

![GitHub stats](https://github-readme-stats.vercel.app/api?username=<username>&show_icons=true)
![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=<username>&layout=compact)

Templates

The layout also needed an update. Luckily, Quarto has five templates to change up the layout. You can change the template using the template option in the YAML header. The five templates are jolla (default), trestles, solana, marquee, and broadside.

---
title: "About Me"
about:
  template: jolla
---

After testing them all, I decided to go with the jolla theme. I like that the photo and title are centered on the page, matching my centered skill icons.
